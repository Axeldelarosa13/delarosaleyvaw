Permisos de Base de Datos Denegados

Tu ERP se ha conectado exitosamente a Firebase, pero la base de datos está bloqueando la lectura y escritura por reglas de seguridad.

Sigue estos 4 pasos exactos para solucionarlo:

  1. Ve a tu Consola de Firebase y entra a tu proyecto.
  2. En el menú lateral izquierdo, haz clic en Firestore Database (Asegúrate de NO elegir "Realtime Database").
  3. En la parte superior, haz clic en la pestaña Reglas (Rules).
  4. Borra absolutamente todo el texto que veas en ese cuadro y pega exactamente el código de abajo:
{`rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}`}

Finalmente, haz clic en el botón azul Publicar. Una vez hecho, recarga esta página y el sistema funcionará al 100%.
